Cloud Sync (Optional)

By default, your friends list is stored in your browser only. Connect a free Supabase project to sync across devices and have a real database backup.

Setup (one-time, ~3 min):
1. Sign up at supabase.com (free)
2. Create a project, then in the SQL editor run:
create table friends ( id bigint generated by default as identity primary key, name text unique not null, email text, birthday text not null, relation text, notes text, source text, created_at timestamp with time zone default now() ); alter table friends enable row level security; create policy "anon all" on friends for all using (true) with check (true); 3. Settings → API → copy your Project URL and anon public key below.

Note: Use the anon public key only -- never the service_role key. The anon key is safe for client-side use because Supabase enforces row-level security on the database.
